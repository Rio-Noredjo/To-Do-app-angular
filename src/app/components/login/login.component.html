Applicatie: To-Do

Het doel van deze applicatie is om mijn skills als full stack java developer te laten zien. De applicatie bestaat uit een Angular front end, springboot backend en een mysql database.
De applicatie bestaat uit verschillende facetten van programmeren. Je kan als gebruiker CRUD toepassen op een Item en op een User. Dit is afhankelijk welke rol je heb als gebruiker.

Pagina: Login

Dit is de inlog pagina van de To-Do app. Zoals als de naam zeg kan je inloggen op deze pagina om de rest van de applicate te bereiken. ZOnder in te loggen is het niet mogelijk om de rest van de applicatie te bekijken. Ook niet door de URL aan te passen. Naast de inloggegevens is er een lijst te vinden met verschillende gebruikers en hun rollen. Elke rol heeft andere rechten.
