To-Do application

The purpose of this application is to show my skills as a full-stack Java developer. The application consists an Angular front end, Spring Boot back end and a MySQL database. To gain technical insight into the application, I recommend viewing the code. By looking at the code you'll get a good idea of my coding skills. The purpose of this application is to show my skills on the different facets of programming, like:

  • Communication between front end, back end and database
  • Apply CRUD to objects
  • Using forms with validation
  • HTML, CSS and Bootstrap
  • Unit tests
Login

This is the login page of the application. As the name already tells you, you can login here to gain access to the application. Without logging in, it is not possible to view the rest of the application. This is not even possible by changing the URL. In addition to the login form, there is a list of different users. These users are in the database. Please choose a user from the list to login.
